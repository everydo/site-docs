润普为 网易 提供Zope/Plone技术培训，并帮助网易使用Plone构建 网易企业知识门户系统 。该系统涵盖了产品开发、公司和部门门户、项目管理、知识管理等众多方面，成为网易在线娱乐事业部的主要工作系统之一。
系统于2006年1月开始小范围推广，于2006年4月全面上线，系统用户超过1,000人。

---------------------------------

**系统主要功能包括：**

•	项目门户 

•	产品开发门户 

•	部门/小组门户 

•	邮件列表 

•	跟踪器 

•	文件库 

•	论坛 

•	知识库 

•	问卷 

•	企业博客(Blog) 

网易公司自身在系统上扩展了图书借阅、招聘管理、系统监视等模块。

**系统组成：**

•	Zope 2.8.x 

•	Plone 2.1.x 

•	操作系统： Debian 

•	LDAP服务器：OpenLDAP 

•	邮件列表服务器：Exim4 

•	Web服务器：Apache2 + SSL
