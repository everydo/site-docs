标签搜索
==============================
可以为文件打上对应标签进行分类；可通过标签搜索到相应的文档


系统搜索规则是怎样？
==============================
汉字可独字搜索;英文和数字可分开搜索，支持使用通配符(*)进行搜索，例如：对于“A2428-装配.pdf”，如输入“A2428”,“A2*”等都可搜索到


搜索结果
=============================
搜索结果支持对关键字加亮，会提取文件相应内容显示(方便查看搜索结果，找到需要的文件)
