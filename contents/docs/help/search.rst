系统搜索规则是怎样？
==============================
和 Google 的搜索一样，对于有下划线“_”连接的，需要整体搜索；英文和数字必须全部连在一起搜索，汉字可独字搜索，英文按照单词为最小搜索单位的（英文和数字都是的），例如：对于“A2428-装配.pdf”，系统认为“A2428”和“pdf”都是一个整体的英文单词，应该输入整个单词进行搜索，如输入“A2428”，而不可以“A2”。
